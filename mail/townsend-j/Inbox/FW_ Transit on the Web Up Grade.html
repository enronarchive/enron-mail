Transit on the Web is upgrading its mainframe operating system.
Customers, who have not done so already, will need to upgrade
their version of QWS3270 from 16-bit to 32-bit. QWS3270 is the
tool used for Confirmation Ranking, ONE Contracting and Capacity
Release.

We are asking that all of our Transit customers upgrade to
the 32-bit version by February 7, 2002. The Transit Home Page
contains a link with directions to determine your version (16-bit
or 32-bit). It also contains necessary information to upgrade if
needed.

If the upgrade is not completed by February 7, 2002, customers may
not be able to access Confirmation Ranking, ONE Contracting and
Capacity Release.

If you have any questions or if you are a customer using Windows
NT, please call the Transit Help line.
